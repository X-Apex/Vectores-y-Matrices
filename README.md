# 📘 Ejercicios de Vectores y Matrices

Este repositorio contiene una colección de **14 ejercicios prácticos** desarrollados en **C++**, orientados a reforzar conceptos básicos de programación con **vectores y matrices**.  

Cada ejercicio está acompañado de un código funcional y comentado, pensado para estudiantes y principiantes que desean practicar lógica y estructuras de datos.

---

## 📂 Lista de Ejercicios

1. Calcular la media de los elementos de un vector.  
2. Calcular la suma al cuadrado de las componentes de un vector.  
3. Calcular el producto escalar de dos vectores.  
4. Multiplicar un número por un vector.  
5. Sumar un número real a cada componente de un vector.  
6. Encontrar el mínimo de los elementos de un vector.  
7. Encontrar el máximo de los elementos de un vector.  
8. Buscar el menor y el mayor elemento en una matriz de *m x n*.  
9. Sumar los elementos de cada fila de una matriz.  
10. Sumar los elementos de cada columna de una matriz.  
11. Sumar los elementos de la diagonal principal (izquierda a derecha).  
12. Sumar los elementos de la diagonal secundaria (derecha a izquierda).  
13. Copiar los elementos de los bordes de una matriz en sentido horario a un vector.  
14. Sumar los elementos que forman la letra **N** dentro de una matriz.
